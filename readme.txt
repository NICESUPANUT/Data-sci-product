2.project-coproduct.ipynb: เป็นไฟล์ Jupyter Notebook สำหรับโครงการ coproductเนื้อหาในโน้ตบุ๊คนี้ประกอบด้วยการวิเคราะห์และดำเนินการที่เกี่ยวข้องกับข้อมูลใน products.csv
3.project-recomentation.ipynb:เป็นไฟล์ Jupyter Notebook สำหรับโครงการ recommendation เนื้อหาในโน้ตบุ๊คนี้ประกอบด้วยการสร้างและทดสอบระบบการแนะนำสินค้า โดยใช้ข้อมูลใน products.csv
วิธีการใช้งาน
1.เปิดไฟล์ Jupyter Notebook:
คุณสามารถเปิดไฟล์ .ipynb ด้วยโปรแกรม Jupyter Notebook หรือ JupyterLab
รันเซลล์ทีละเซลล์ตามลำดับ เพื่อทำตามขั้นตอนที่กำหนดในโน้ตบุ๊ค

2.เตรียมสภาพแวดล้อม:
ตรวจสอบว่ามีการติดตั้งไลบรารีที่จำเป็นทั้งหมด (เช่น pandas, numpy, scikit-learn, matplotlib เป็นต้น)
คุณสามารถใช้คำสั่ง !pip install <library-name> ในเซลล์ Jupyter เพื่อทำการติดตั้งไลบรารีที่ขาด

3.โหลดข้อมูล:
ตรวจสอบให้แน่ใจว่าไฟล์ products.csv อยู่ในไดเรกทอรีเดียวกันกับไฟล์โน้ตบุ๊ค เพื่อให้สามารถโหลดข้อมูลได้อย่างถูกต้อง
ใช้คำสั่ง pd.read_csv('products.csv') เพื่อโหลดข้อมูลเข้าไปใน DataFrame

4.รันโค้ด:
ทำตามขั้นตอนและคำแนะนำที่ระบุไว้ในโน้ตบุ๊คแต่ละไฟล์
หากพบข้อผิดพลาดหรือปัญหา ให้ตรวจสอบการติดตั้งไลบรารีและการโหลดข้อมูลอีกครั้ง

5.วิเคราะห์ผลลัพธ์:
ตรวจสอบและวิเคราะห์ผลลัพธ์ที่ได้จากการรันโค้ด
สามารถปรับแต่งและทดลองเพิ่มเติมตามความต้องการ

ผลลัพธ์
project-coproduct.ipynb:
-การวิเคราะห์ข้อมูลสินค้า: แสดงผลลัพธ์ที่ได้จากการวิเคราะห์ข้อมูล เช่น จำนวนสินค้าที่แตกต่างกันในฐานข้อมูล การกระจายตัวของสินค้าตามประเภทต่าง ๆ หรือการขายสินค้าในช่วงเวลาที่แตกต่างกัน
-การหาความสัมพันธ์ระหว่างสินค้า: ผลลัพธ์ที่ได้จะแสดงถึงความสัมพันธ์ระหว่างสินค้าที่ผู้ใช้มักจะซื้อพร้อมกัน เพื่อใช้ในการจัดกลุ่มหรือแนะนำสินค้าในอนาคต
-การสร้างกราฟและการแสดงผล: ผลลัพธ์จะเป็นกราฟและตารางที่ช่วยให้เห็นภาพรวมของข้อมูลและการวิเคราะห์ที่ได้ทำไป

project-recomentation.ipynb:
-การเตรียมข้อมูลสำหรับการแนะนำสินค้า: แสดงขั้นตอนการทำความสะอาดและเตรียมข้อมูลสำหรับการสร้างระบบแนะนำสินค้า
-การสร้างโมเดลแนะนำสินค้า: ผลลัพธ์จะแสดงถึงประสิทธิภาพของโมเดลที่สร้างขึ้น เช่น ค่าแม่นยำ (accuracy), ค่าความจำเพาะ (precision), ค่าความไว (recall) และค่า F1-score
-การทดสอบระบบแนะนำสินค้า: ผลลัพธ์จะแสดงรายการสินค้าที่ถูกแนะนำสำหรับลูกค้าแต่ละราย และการประเมินประสิทธิภาพของระบบ
